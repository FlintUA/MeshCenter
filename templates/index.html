Help

Quick guidance for the main areas of MeshCenter.

🚀

Getting Started

Connect MeshCenter to the Meshtastic node, wait for the node and chat lists to load, then select a channel or direct message.

💬

Chats

Select a channel or node conversation, enter a message and press Send. Delivery information appears with the message when available.

📡

Nodes

Select a node in the right panel to inspect radio data, position, telemetry and recent activity.

🎥

Camera & Media

Use Camera for live view and captures. Saved images are available in the local Media gallery.

🖥️

System

Review Raspberry Pi status, network information, radio health and system events.

▦

Workspace

Use the bottom Workspace control to show or hide panels, switch theme and enable compact mode on this browser.

More help topics and troubleshooting guidance will be added in future releases.
